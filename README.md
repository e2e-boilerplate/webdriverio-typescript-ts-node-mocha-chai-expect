# WebdriverIO Boilerplate
    
WebdriverIO end-to-end test automation boilerplate, Using TypeScript, ts-node, Mocha, Chai and Expect.
    
## Getting Started
    	1. git clone git@github.com:e2e-boilerplate/webdriverio-typescript-ts-node-mocha-chai-expect.git
    	2. cd webdriverio-typescript-ts-node-mocha-chai-expect
    	3. npm install
    	4. npm run test
